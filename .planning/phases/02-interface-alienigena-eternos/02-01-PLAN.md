# Plan 02-01: Sci-Fi Premium Layout & Glassmorphic Panels

<objective>
Construir a estrutura visual e os estilos da interface do ItudeAgent usando fontes cinematográficas, cores douradas e painéis de controle retráteis translúcidos (glassmorphism) para logs de conversa e seleção de modelos do Ollama.
</objective>

<context>
- A UI deve ser desobstruída para destacar o canvas interativo no centro.
- O CSS vanilla será escrito em `public/style.css` e importado no `index.html`.
- O painel lateral esquerdo mostrará a lista de modelos disponíveis no Ollama em um `<select>` dinâmico.
- O painel lateral direito exibirá o console de logs/transcrições do chat em tempo real.
</context>

<tasks>

    - .planning/phases/02-interface-alienigena-eternos/02-CONTEXT.md
  </read_first>
  <action>
    Criar `backend/public/style.css`.
    1. Importar fontes do Google Fonts: 'Cinzel' para títulos e 'Outfit' para textos do corpo.

    4. Adicionar estilos para o dropdown e console de logs.
